Setup guide
1) For each class, publish your Google Slides and copy the presentation ID (the part between /d/ and /edit).
2) Add entries to the configuration with keys as ISO week codes like 2025-W36 and values as the slide IDs.
3) This page auto-selects the current ISO week. If no deck exists for this week, it falls back to the most recent previous week.
4) Optional: create a file at /data/agendas.json to manage the config outside this HTML. If present, it overrides the inline config below.
