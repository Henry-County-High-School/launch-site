Setup guide
1) Publish each Google Slides deck. Use the published ID (portion after /d/e/ and before /pub). Using the file ID (between /d/ and /edit) also works.
2) Add it under the correct ISO week key (e.g. 2025-W36).
3) Page auto-selects the current ISO week if present, else the most recent prior.
4) Optionally create /data/agendas.json (same shape as inline config) to update without editing HTML.
5) Share direct links using the Copy Link button (hash stores class & week).
